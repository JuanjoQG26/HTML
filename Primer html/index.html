HTML

Que es?

Es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos
Atajo para que suelte el esquelo inicial: ! tab

Elementos principales

Doctype --> Indica que es tipo HTML 5, el explorador lo identifica como pagina web
Html --> Etiqueta raiz de todo html
Head --> Caracteristicas que va a tener la pagina, no necesariamente visible
Body --> Donde se coloca todo el cuerpo / contenido de la pagina
Title --> Etiqueta del titulo de la pestaña

Otros elementos

Encabezados

Existen diferentes tipos de encabezados en HTML que van desde
la etiqueta h1 hasta la h6. Siendo h1 la de mayor jerarquia y h6 la de menos jerarquia

Formas de escritura

Tambien esta el hr que esta para separar una seccion de otra
La etiqueta p que es para escribir parrafos
La br que es para saltos de linea en medio de parrafos
